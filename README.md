=============================================
  Google PAA Scraper - Mac Kurulum
=============================================

1. Bu klasörü istediğin yere koy (örn: Masaüstü)

2. "Hasar görmüş" hatası alırsan Terminal'i aç ve şunu yaz:

   xattr -cr ~/Desktop/PAA\ Scraper\ Mac/

   (Klasörü başka yere koyduysan yolu ona göre değiştir)

3. "PAA Scraper.command" dosyasına çift tıkla

4. İlk açılışta otomatik kurulum yapar (1-2 dakika):
   - Python bağımlılıklarını yükler
   - Chromium tarayıcı indirir

5. Tarayıcı otomatik açılır, kullanmaya başla!

NOT: Python 3 gereklidir. macOS 12+ sürümlerde zaten yüklüdür.
     Yoksa: https://www.python.org/downloads/
